# spicyipsum

Lorem ipsum generator with a nip.

## INSTALLATION

### install system dependencies

- git
- npm
- nodejs
- sqlite3
- make

### install project

```
git clone git@github.com:booksmith-io/spicyipsum.git
cd spicyipsum
npm run setup
cp -a .spicyipsumrc.example .spicyipsumrc
```

### run the project

```
npm run development
```

### update project

```
npm run upgrade
npm run update_deps
npm run upgrade_database
```
